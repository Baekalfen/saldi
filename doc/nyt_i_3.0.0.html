Kære Saldi-bruger.


Saldi er blevet opdateret.

Finans -> kassekladde.  
  • Ved klik på tomt modpost felt til debitor eller kreditorpostering vises sidste 5 posteringer.
  • Kommentarfeltet kan redigeres efter bogføring af kassekladden.
Finans -> regnskab.  
  • Indeholder nu budget. Sidste års tal kan indsættes med tillæg af selvvalgt procentsats.
Finans -> rapport.  
  • Har fået en ny rapport - Rasultat / Budget som viser realiserede tal, budget tal og afvigelse i %.  
Kasseløsning. 
  • Saldi indeholder nu en integreret kasseløsning, som understøtter touchskærm, bonprinter, stregkodescanner & kasseskuffe.
  • Se mere på http://saldi.dk/saldikasse
Debitor -> konti. 
  • Debitoroversigten er blevet ændret så man nu selv kan bestemme hvilke felter som skal medgå på oversigten
  • På debitorkortet er kommet et kategori felt, så så debitorer kan være tilknyttet flere kategorier.
System -> indstillinger. 
  • Der er blevet tilføjet en rabatmatrix så de enkelte kundegrupper kan opsættes til individuel rabat for de enkelte varegrupper.

Herudover er momsstrukturen blevet ændret så momsen på varegrupper styres efter kontoplanen.
Dette har især betydning for dem som handler med både kunst og normale varer.
